# 如何在iPhone搜索引擎上找到你的网站

## 📱 iPhone上的搜索引擎

iPhone默认使用Safari浏览器，其搜索引擎通常是：
- **Google**（全球大部分地区）
- **百度**（中国大陆地区）
- **必应（Bing）**（部分地区）

## 🔍 让别人在iPhone上搜索到你的网站

### 方法一：部署网站并提交给搜索引擎（必须）

要让网站在iPhone的搜索引擎中显示，需要：

#### 步骤1：部署网站

**选择免费托管平台：**

**A. GitHub Pages（推荐）**
1. 注册GitHub账号：https://github.com
2. 创建仓库并上传网站文件
3. 启用GitHub Pages
4. 获得网址：`https://你的用户名.github.io/项目名`

**B. Netlify（最简单）**
1. 访问：https://www.netlify.com
2. 注册账号
3. 拖拽网站文件夹上传
4. 获得网址：`https://项目名.netlify.app`

#### 步骤2：提交给Google（全球用户）

**在iPhone上操作：**
1. 用iPhone浏览器访问：https://search.google.com/search-console
2. 点击"添加属性"
3. 输入你的网站网址（例如：`https://ilovehzl.github.io/website`）
4. 选择验证方式（推荐：HTML文件验证）
   - 下载验证文件
   - 上传到网站根目录
   - 回到Google Search Console点击"验证"
5. 验证成功后，提交sitemap：
   - 左侧菜单选择"Sitemap"
   - 输入：`https://你的网址/sitemap.xml`
   - 点击"提交"

**等待时间：**
- Google索引通常需要1-4周
- 提交后几天内可能开始显示

#### 步骤3：提交给百度（中国大陆用户）

**在iPhone上操作：**
1. 用iPhone浏览器访问：https://ziyuan.baidu.com
2. 注册/登录百度账号
3. 点击"添加网站"
4. 输入网站网址
5. 选择验证方式（推荐：HTML文件验证）
6. 验证成功后，提交sitemap

#### 步骤4：确保网站可被爬虫访问

✅ 确保网站是公开的（Public）
✅ 确保robots.txt允许搜索引擎访问
✅ 确保sitemap.xml可访问
