## 部署步骤

### 1. 购买域名（推荐）

**推荐域名：**
- `ilovehzl.com`（首选）
- `i-love-hzl.com`
- `ilovehzl.net`

**购买平台：**
- **Namecheap**：https://www.namecheap.com（国外，约$10/年）
- **GoDaddy**：https://www.godaddy.com（国外，约$12/年）
- **阿里云**：https://wanwang.aliyun.com（国内，约¥50/年）

### 2. 部署网站

#### 选项A：GitHub Pages + 自定义域名
1. 注册GitHub账号
2. 创建仓库上传文件
3. 启用GitHub Pages
4. 在仓库设置中添加自定义域名 `ilovehzl.com`
5. 在域名DNS设置中添加GitHub的CNAME记录

#### 选项B：Netlify + 自定义域名
1. 访问 https://www.netlify.com
2. 拖拽文件夹上传
3. 在Domain settings中添加自定义域名
4. 配置DNS（按Netlify提示操作）

### 3. 提交给搜索引擎

#### Google搜索
1. 注册Google Search Console：https://search.google.com/search-console
2. 添加网站属性
3. 提交sitemap.xml

#### 百度搜索
1. 注册百度站长平台：https://ziyuan.baidu.com
2. 添加网站
3. 提交sitemap.xml

#### 小红书/抖音（如果适用）
- 可以在这些平台分享网站链接

### 4. 更新sitemap.xml

部署后，需要将 `sitemap.xml` 中的 `https://your-domain.com` 替换为你的实际域名。

3. **社交媒体**：定期分享和推广网站
4. **持续更新**：定期更新网站内容
